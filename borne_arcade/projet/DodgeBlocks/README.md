# DodgeBlocks

Jeu Python/Pygame: survivre le plus longtemps possible en evitant les blocs qui tombent.

## Nouveautes

- Score sauvegarde localement (`highscore.json`).
- Difficulte progressive avec le temps:
  - spawn plus frequent,
  - blocs plus rapides,
  - acceleration continue,
  - plus de blocs simultanes.
- Musique de fond integree: `assets/supra_mario.wav`.

## Lancer

Depuis la racine du projet `borne_arcade`:

```bash
./DodgeBlocks.sh
```

## Controles

- Gauche / Droite: deplacement
- A / Space / Enter: start / restart
- ESC: quitter
